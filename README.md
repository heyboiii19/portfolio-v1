# Aditya Tiwari Portfolio

A modern personal portfolio website showcasing my projects, skills, internships, certifications, and journey in Bioinformatics, AI, and Full-Stack Development.

## 🚀 Features

* Responsive design
* Modern dark theme UI
* Smooth animations and transitions
* Skills, experience, and project showcase
* Resume download functionality
* Contact section with social links

## 🛠️ Built With

* HTML5
* CSS3
* Bootstrap 5
* JavaScript
* AOS (Animate On Scroll)
